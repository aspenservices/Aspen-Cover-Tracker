# Aspen Covers Tracker

Internal dashboard for tracking spa cover inventory at Aspen Spas Service Department.

**Live URL**: https://aspenservices.github.io/Aspen-Cover-Tracker/

## Tech stack

- Single-file HTML + vanilla JavaScript
- Firebase Realtime Database (invoice-daily-f170e)
- SheetJS for Excel I/O
- GitHub Pages hosting

## Integrations

- TECH-TICKETS Service Log (auto-create on arrival)
- Invoice Tracker (shared customer DB)

## Maintained by

Service Department · Aspen Spas · St. Louis, MO
